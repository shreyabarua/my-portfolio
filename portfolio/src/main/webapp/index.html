Projects & Contributions

Automatcher

May 2019 - August 2019

What is Automatcher?

As an intern on Google's C++ Libraries team, my podmate and I worked on a tool called Automatcher . Automatcher is an AST matcher tool which generates clang queries from user-provided code snippets. Usually complex queries need to be written out by hand, but Automatcher automates this process.

What I did:
  • Took ownership of the "hint" based approach to solving the problem of generating an Abstract Syntax Tree (AST) matcher from a user-friendly specification.
  • Implemented support for three different methods of annotating code fragments to guide generation of the AST matcher using C++.
  • Led the design and implementation of the first version of the essential data structure for representing "choice" in a code description.
